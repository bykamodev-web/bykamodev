# bykamodev-web

[bykamo.dev](https://bykamo.dev) のソースコード。

AI実装、自動化、プロダクト設計、開発ログ、実験記録を公開する開発者 KAMO のサイト。

## Tech Stack

- [Astro](https://astro.build/) — Static Site Generator
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [microCMS](https://microcms.io/) — Headless CMS
- [Cloudflare Pages](https://pages.cloudflare.com/) — Hosting

## Development

```bash
pnpm install
cp .env.local.example .env.local  # microCMS の API キーを設定
pnpm dev
```
